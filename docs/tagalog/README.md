# Mga Dokumentasyon sa Tagalog/Filipino

Ang folder na ito ay naglalaman ng mga salin sa Tagalog ng mga webinar materials para sa DevOps series.

## Mga Available na Webinar

1. **[Webinar 1 — Docker + CI/CD gamit ang GitHub Actions](./webinar-1-docker-cicd.md)**
   - Panimula sa Docker containerization
   - Pag-build at pag-run ng Docker images
   - Docker Compose para sa multi-service applications
   - CI/CD automation gamit ang GitHub Actions

2. **[Webinar 2 — AWS Deployment gamit ang EC2, Networking, at HTTPS](./webinar-2-aws-deployment.md)**
   - Pag-setup ng AWS EC2 instances
   - Pag-configure ng Security Groups at Elastic IP
   - Pag-deploy ng Docker containers sa cloud
   - Pag-secure ng applications gamit ang HTTPS

## Mga Tandaan

- Ang mga technical terms tulad ng "Docker", "CI/CD", "Git" ay pinanatiling English dahil ito ay karaniwang ginagamit sa industriya
- Ang mga code blocks at commands ay hindi isinalin at pinanatiling English
- Ang dokumentong ito ay optimized para sa mga Filipino developers na mas komportable sa Tagalog
